## graphify

This project can optionally use a Graphify knowledge graph at `graphify-out/`.

Rules:
- If `graphify-out/GRAPH_REPORT.md` exists, read it before answering architecture or graph questions.
- If `graphify-out/wiki/index.md` exists, navigate it instead of reading raw files.
- This public repo intentionally excludes private corpora and generated graph outputs.
